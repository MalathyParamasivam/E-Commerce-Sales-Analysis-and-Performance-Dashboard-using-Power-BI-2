# E-Commerce-Sales-Analysis-and-Performance-Dashboard-using-Power-BI-2
This assignment demonstrates the use of Power BI for analyzing E-Commerce sales data through DAX calculations and interactive visualizations. Key business metrics such as sales, profit, targets, and order performance were analyzed to generate meaningful insights and support data-driven decision-making

## Project Description
This project analyzes E-Commerce sales data using Power BI. DAX calculated columns and measures were created to generate key business metrics such as Order Count, Average Profit, and Year-to-Date Sales. Interactive visualizations were developed to analyze sales performance, profitability, sales targets, and regional trends.

## Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)

## Features
- Calculated Columns and Measures
- Sales Target Achievement Analysis
- Monthly Sales Trend Analysis
- Profitability Analysis
- Geographic Sales Analysis
- Interactive Dashboard

## Dataset
- List of Orders
- Order Details
- Sales Target

## Key Visualizations
- Clustered Column Chart
- Donut Chart
- Line Chart
- Scatter Chart
- KPI Cards
- Matrix
- Map
- Treemap
- Funnel Chart

## Outcome
The dashboard provides meaningful business insights and supports data-driven decision-making through effective data visualization and performance analysis.
